**一句话介绍：** 把窗口拖拽变成磁铁吸附，靠近边缘自动对齐、贴边隐藏。

## ✨ 核心功能（默认操作方式）

- **智能吸附**：拖拽窗口时自动对齐到其他窗口边缘或屏幕边界，带幽灵预览框。
- **多功能菜单**：按 `Win + Z` ,呼出多功能菜单，百来项窗口功能可选择使用。
- **任意位置拖拽**：无需点标题栏，按住 `左Win + 鼠标左键` 或鼠标侧键即可拖移。
- **任意位置调整大小**：按住 `右Win + 鼠标左键` 并在窗口内任意位置拖动，即可调整窗口尺寸。
- **临时禁用**：拖拽时按住 `Shift` 可临时跳过吸附。
- **窗口联动**：贴边排列的窗口，按住 `Ctrl` 拖动其中一个，其余整体跟随移动。
- **智能同步**：吸附时按住 `Alt` 可自动匹配目标窗口的宽/高。
- **贴边隐藏**：将窗口拖到屏幕边缘并按住 CapsLock（默认），窗口会自动收缩只剩一条凸起边缘；鼠标悬停时滑出，移开后再次隐藏。
- **高度可定制**：所有开关和快捷键均可通过 `GhostSnap.ini` 配置文件调整。

## ❓ 常见问题  
- **Q: 我不想每次拖拽窗口都自动吸附，只想在按住某个键时才触发吸附，怎么设置？**

  A: 在 设置中心 的基础吸附中，勾选 反向模式。

- **Q: 个别软件（如任务栏、截图工具）我不想让它被吸附？**

  A: 在 设置中心 的 黑名单中添加对应窗口规则（支持标题/类名/进程名），一行一个。已内置常见工具的黑名单。

- **Q: 幽灵窗口太显眼？**

  A: 在 设置中心 的 外观特效 中 可修改（透明度，0-255）或设置 关闭幽灵窗口特效。

- **Q: 联动移动没反应？**

  A: 确保窗口之间是贴边排列的，且拖拽时持续按住 Ctrl 键。可在托盘菜单中确认功能已开启。

- **Q: 某些程序无法被拖拽或吸附（例如高权限软件）？**

  A: 部分以管理员身份运行的程序（如任务管理器、安装向导）可能无法被普通权限的脚本操作。遇到这种情况，可以右键脚本 → 以管理员身份运行，重新加载后即可对这些窗口生效。
