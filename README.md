# 窗口吸附增强工具 (WindowSnap Enhancer)

**一句话介绍：** 把窗口拖拽变成磁铁吸附，靠近边缘自动对齐。

---

## ✨ 核心功能

- **智能吸附**：拖拽窗口时自动对齐到其他窗口边缘或屏幕边界，带幽灵预览框。
- **任意位置拖拽**：无需点标题栏，按住 `左Win + 鼠标左键` 或鼠标侧键即可拖移。(可自定义按住键或禁用)
- **临时禁用**：拖拽时按住 `Shift` 可临时跳过吸附。
- **窗口联动**：贴边排列的窗口，按住 `Ctrl` 拖动其中一个，其余整体跟随移动。(可自定义按住键或禁用)
- **Alt 智能同步**：吸附时按住 `Alt` 可自动匹配目标窗口的宽/高。
- **高度可定制**：所有开关和快捷键均可通过 `GhostSnap.ini` 配置文件调整。

---

## ⌨️ 默认操作方式

| 操作 | 效果 |
|------|------|
| 普通拖拽标题栏 | 自动吸附（20px 距离触发） |
| 拖拽时按住 `Shift` | 禁用吸附，自由移动 |
| `左Win + 鼠标左键` 拖拽任意位置 | 不点标题栏直接移动窗口 |
| 鼠标侧键 (`XButton1`) 拖拽 | 同上（可改成其他按键） |
| 按住 `Ctrl` 拖拽已贴边的窗口 | 联动相邻窗口一起移动 |
| 吸附时按住 `Alt` | 同时匹配目标窗口的高/宽 |
